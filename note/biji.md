#课堂笔记
####github命令
cd..返回上一级目录
cd 目录名 进下一级目录
ls 查看当前目录下的文件
ll 查看目录详细信息
d f:进入f盘
####github上传文件
git init 初始化一个git仓库
git status 查看文件状态
git add .  把编辑区放在暂存区
git commit -m “  ”   将暂存区文件提交到分支区
git push origin master 同步文件
####快捷键
shift+insert 粘贴
####文件命名格式
username.github.io
